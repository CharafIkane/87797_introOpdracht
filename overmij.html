Ervaring

Op het Grafische Lyceum leerde hij bij het begin zijn eerste stappen in web development talen zoals: HTML, CSS en Javascript. Verder heeft hij ook leren werken met Ubuntu om daar via de terminal velen installaties te volgen. Hij heeft inmiddels ook deels ervaring met het opzetten van servers en hoe zo een server kan werken. Later werkte hij ook met andere talen zoals C# en PHP. Hij vond PHP een hele bijzondere taal, omdat hij het heel interessant vinden om dingen in de database te zetten en dat daarna te kunnen nalezen.
Hier zie je een 'progressbar' over hoe ver zijn ervaring is met elke codeertaal / library.
